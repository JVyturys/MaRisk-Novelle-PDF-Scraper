# MaRisk-Novelle-PDF-Scraper
Automatisierte MaRisk-Gap-Analyse: Das Tool extrahiert regulatorische Texte aus BaFin-PDFs, gleicht sie mit Bestandsfassungen ab und visualisiert Änderungen per Diff-Anzeige. Inkl. Dashboard zur Zuweisung von Status, Teams und Prioritäten sowie CSV-Export für Maßnahmenpläne. Ideal für Compliance &amp; Audit.
